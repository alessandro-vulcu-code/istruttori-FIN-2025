# Teoria del movimento

## Capacità motorie
**Cosa sono?** sono i presupposti delle azioni motorie, ossia le potenzialità di un individuo.
**Quante sono?** si dividono in due categorie:

### Capacità coordinative
Sono legate al **sistema nervoso** - lato cognitivo del movimento.
Sono divise in **generali** e **speciali**.

#### Capacità coordinative generali
Sono tre:
1) Capacità di **apprendimento**: prima imparo il movimento.
	1) **Coordinazione grezza**: prima esecuzione imperfetta, migliora ripetendo correttamente il movimento.
	2) **Coordinazione fine**: il movimento diventa sempre più dettagliato, preciso, efficace.
	3) **Disponibilità variabile**: ultima fase, il movimento viene ulteriormente affinato e può essere eseguito in qualsiasi condizione.
2) **Capacità di controllo**: precisione esecutiva di una determinata abilità.
3) **Capacità di adattamento e trasformazione**: adattare il movimento alle diverse situazioni.

#### Capacità coordinative speciali

| Tipo                         | Descrizione                                                                                                |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------- |
| Ritmo                        | scelta temporale degli impulsi per realizzare in modo adeguato un'azione finalizzata                       |
| Reazione                     | elaborazione rapida dello stimolo percettivo $\Rightarrow$ velocità di trasmissione degli organi recettori |
| Equilibrio                   | mantenere postura utile all'azione (vestibolare)                                                           |
| Orientamento                 | mantenere il giusto rapporto con il campo d'azione e i riferimenti spaziali                                |
| Combinazione motoria         | -                                                                                                          |
| Differenziazione cinestesica | dosare forza per massima efficacia                                                                         |
| Anticipazione motoria        | previsione, pre-costituzione dell'azione successiva                                                        |
| Fantasia motoria             | usare abilità apprese per risolvere un problema motorio                                                    |
